# springAiProject
Curso prático sobre integração de IA com Java, Spring Boot e Spring AI.
Criação de APIs inteligentes conectadas ao ChatGPT/OpenAI, geração de imagens com DALL·E e transcrição de áudio em texto.
Também aborda DeepSeek, Ollama, React JS, Postman, MCP e boas práticas para projetos de portfólio.
